# Unsupervised Learning: Clustering Techniques

## Project Overview

This project was completed as part of my AI & ML Internship (Week 4 - Task 1). The objective of this project is to understand the basics of unsupervised learning by implementing different clustering algorithms using Python and Scikit-learn.

## Algorithms Implemented

- K-Means Clustering
- Hierarchical Clustering
- DBSCAN Clustering

## Dataset Used

- Iris Dataset (Scikit-learn Built-in Dataset)

## Project Files

- `notebooks/unsupervised_learning_clustering_techniques.ipynb` – Jupyter Notebook containing the implementation of clustering algorithms.
- `docs/summary_report.pdf` – Summary report of the project.
- `data/dataset_source.txt` – Information about the dataset used.
- `requirements.txt` – List of required Python libraries.

## Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Conclusion

This project demonstrates the implementation and comparison of three clustering algorithms using the Iris dataset. It provides a beginner-friendly introduction to unsupervised learning and clustering techniques.
